# Agentに情報を与えよう

## 実行コマンド

```
# APIサーバ(Python)の起動
uv run python -m todo_api.server.api

# フロントエンドサーバ(React)の起動
cd todo_frontend && npm start
```

## 情報を与えない場合

```
タスクを変更する機能を実装して
```

たくさんファイルを読んでやり遂げる。

目的を達成したら止まる。

## 情報を与えた場合

インストラクションファイルを生成する。

そして同じタスクを実行する

```
タスクを変更する機能を実装して
```

テストまでやり遂げるように頑張るようになる。

## ブラウザを使って情報を与える

コマンド "Simple Borser: Show" を使用して、`http://localhost:3000` を開く。

タスクの入力欄を指示して、横幅を広げるように指示する。
